# 📚 BookList App (Android)

A simple Android application built with **Kotlin** that displays a list of books with search functionality. Users can browse books and view detailed information about each book, including description and cover image.

---

## ✨ Features

- 📖 Display list of books
- 🔎 Search books by title
- 📄 Book details screen with:
  - Description
  - Cover image
- 🔄 Smooth navigation between list and detail screens
- ⚡ Clean MVVM architecture
- 💉 Dependency Injection with Koin
- 🌐 Networking with Ktor

---

## 📱 Screens

### 🏠 Book List Screen
- Shows a list of available books
- Supports real-time search filtering
- Click on any book to view details

### 📄 Book Detail Screen
- Displays selected book information
- Shows book cover image
- Displays full description fetched from API

---

## 🏗️ Tech Stack

- **Kotlin**
- **MVVM Architecture**
- **Jetpack Navigation Component**
- **Koin (Dependency Injection)**
- **Ktor Client (Networking)**
- **StateFlow (Reactive State Management)**
- **XML Views**
